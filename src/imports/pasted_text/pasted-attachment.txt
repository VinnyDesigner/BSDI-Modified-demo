Fix Spatial Mode Drawing Tools

When the Spatial tab is active, enable full map drawing functionality.

Drawing Tools to Enable

Show a vertical drawing toolbar on the right side of the map with the following tools:

Polygon Draw

Rectangle Draw

Circle Draw

Extend Boundary

Delete Selected Area

Clear All Selection

Drawing Behavior

Admin should be able to draw boundaries directly on the map.

The selected boundary should appear with:

BSDI red border

Semi-transparent red fill

Rules

Prevent drawing outside allowed map extent.

If a new shape overlaps an existing shape → merge shapes automatically.

Allow editing vertices of existing polygons.

Allow extending existing boundaries.

Show a role label on the map.

Example:

Editing Role: Electricity Authority

Show Update button at the top-right corner of the map container.

Button style:

BSDI red

Rounded

Save icon

Function:

Saves the drawn spatial boundary.

Assigns the boundary to the selected role.

Alignment Rules

Floating UI elements must have z-index priority above map tiles but below system alerts.

Avoid stacking multiple panels on the same position.
